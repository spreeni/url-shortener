URL Shortener

This is a URL shortener. Enter your query and if it is a valid unix code or a date the app will return a JSON object with the query-date in unix code as well as a natural date.

Example entry of an url into the database:

https://icy-detail.glitch.me/new/https://www.google.com/

Example output:

{"original_url":"https://www.google.com/","shortened_url":"https://icy-detail.glitch.me/3"}

Example usage:

https://icy-detail.glitch.me/3/

Will redirect to:

https://www.google.com/
